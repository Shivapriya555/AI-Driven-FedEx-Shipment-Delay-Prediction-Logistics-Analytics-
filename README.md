# AI-Driven FedEx Shipment Delay Prediction & Logistics Analytics

## Project Overview
This project analyzes FedEx shipment logistics data and builds a machine learning model to predict shipment delays using Python and Machine Learning techniques.

## Model Accuracy
Achieved 93.13% prediction accuracy using Logistic Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Correlation Heatmap Analysis
4. Feature Engineering
5. Machine Learning Model Building
6. Model Evaluation using Confusion Matrix

## Key Features
- Shipment delay prediction
- Logistics trend analysis
- Correlation analysis
- Data visualization
- Predictive analytics
